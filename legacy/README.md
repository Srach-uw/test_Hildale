# Legacy Results (SUPERSEDED)

These files come from the earlier generation of this project
(https://github.com/Srach-uw/Shreshth_Hildale_Project) and are kept only for
provenance and before/after comparison. **Do not use them as current results.**

Why they are superseded:

- They were produced from an older canonical sample of **1716 planets**;
  the current canonical sample is **2474 planets** with a rebuilt disk
  classification and posterior inventory (see `../docs/`).
- The per-planet eccentricity method used there was shown by injection tests
  (see the old repo's REPORT.md) to be biased by wide posteriors; the current
  effort replaces those inputs with ALDERAAN transit-shape posteriors.

Contents:

| file | what it was |
|---|---|
| `REPORT.md` | narrative report of the earlier replication attempt |
| `final_results.csv` | population eccentricity fits from the 1716-planet sample |
| `table2_bayes.csv` | Bayesian model comparison from the same sample |
| `figures/` | figures generated by the earlier pipeline |
